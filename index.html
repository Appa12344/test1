What isSVG Gradient?

A gradient is a smooth transition from one color to another. In addition, several color transitions can be applied to the same element.


There are two main types of gradients in SVG:
1.Linear
2.Radial

SvglinearGradient-< linearGradient>

The linearGradient element is used to define a linear gradient.

The linearGradient element must be nested within a defs tag.
The defs tag is short for definitions and contains definition of special elements (such as gradients).

Linear gradients can be defined ashorizontal, vertical or angular gradients:

1.Horizontal gradients are created when y1 and y2 are equal and x1 and x2 differ
2.Vertical gradients are created when x1 and x2 are equal and y1 and y2 differ
3.Angular gradients are created when x1 and x2 differ and y1 and y2 differ
